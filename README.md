# tokenlens-design
Static TokenLens design explorations and route mockups for marketing, authentication, onboarding, and dashboard surfaces.

## Current marketing surfaces

- `landing.html`: docs-aligned marketing home focused on LLM usage observability, early SDK integration, dashboard previews, FAQ, and clear Getting Started / Pricing / Docs CTAs.
- `pricing.html`: shared-CSS pricing page covering Free, Pro, and Team plans, pricing dimensions, FAQ, and final CTA.
- `about.html`: optional trust/context page kept coherent with the marketing navigation and CTA language.
- `styles/design-system.css`: shared stylesheet for touched marketing pages.

## Docs linkage

Marketing pages use relative links into the sibling `../tokenlens-docs` repository for docs-oriented CTAs during local review.

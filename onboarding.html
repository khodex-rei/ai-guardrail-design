Onboarding

Get from signup to first event received

The fastest path is create workspace, create project, generate an API key, install the SDK, send a test event, and confirm data appears.

Activation checklist

Authenticated users with completed onboarding should go to /app.

3 of 6 complete
  • 1
    Create workspace
    Workspace name and billing region are ready.
  • 2
    Create project
    Primary API surface selected.
  • 3
    Generate API key
    Key displayed once and copied.
  • 4
    Install the SDK
    Choose language and copy the install command.
  • 5
    Send a test event
    Call the wrapped client from your app or local script.
  • 6
    Confirm first event received
    Once data arrives, continue to the project dashboard.
